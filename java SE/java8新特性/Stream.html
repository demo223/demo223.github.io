# Stream

在Java 8中，得益于Lambda所带来的函数式编程， 引入了一个全新的Stream流概念。

目的：用于简化集合和数组操作的API

# Stream的使用

# Stream流的获取

要想使用Stream操作集合和数组第一步就要先获取Stream流

  • 集合获取Stream流

    1. Collection集合(List,Set集合等)获取Stream流

      Collection<String> list = new ArrayList<>();
Stream<String> listStream = list.stream();

    2. Map集合获取Stream流

      Map<String， Integer> map = new HashMap<>();

Stream<Integer> valueStream = map.values().stream();
// 获取Map集合中所有的键值对
Stream<Map.Entry<String，Integer>> entryStream = map.entrySet().stream

# Stream流的常见用法

Stream流的方法分为终结方法和非终结方法

  • 终结方法 流不会在继续使用
  • 非终结方法(中间方法) 会产生新的流，可以对新的流进行链式操作
操作 方法
对流中的数据进行过滤，并返回一个子流对象 filter()
将流中的元素按规则映射到另一个流中 map()
删除流中重复的元素 distinct()
丢弃流中前n个元素 skip(long n)
截取流中元素的长度(如果大于流的元素个数则返回所有元素，不会报错) limit(long n)
将两个流对象合并为一个流 Stream.concat(流1，流2)
删除流中重复的元素 distinct()
将流中的元素默认按自然顺序排序(也可以通过自定义比较器来排序) sorted(比较器)
获取Stream流中的元素的个数 count()
获取Stream流中的元素的最大值(根据自定义的比较器) max(比较器)
获取Stream流中的元素的最小值(根据自定义的比较器) min(比较器)
把Stream流中的所有元素聚合成一个结果 reduce(聚合函数)

  10. 使用reduce(聚合函数)将Stream流中的所有元素进行指定操作得到一个结果

```java
// 1.求和
Stream<Integer> stream = Stream.of(1，2，3，4，5);
// 求和方法1
Optional<Integer> sum1 = stream.reduce((sum， a) -> sum += a);
System.out.println(sum1.get());  //  15
// 求和方法2
// java.lang.Integer.sum()是java中的内置方法，该方法返回其参数的总和
Optional<Integer> sum2 = stream.reduce(Integer::sum);
System.out.println(sum2.get()); //  15
// 求和方法3
Integer sum3 = stream.reduce(0， Integer::sum);
System.out.println(sum3);   //  15

// 2.计算乘积
Stream<Integer> stream = Stream.of(1， 2， 3， 4， 5);
// 计算乘积方法1
Optional<Integer> product1 = stream.reduce((a， b) -> a *= b);
System.out.println(product1.get()); //  120
// 计算乘积方法2
Integer product2 = stream.reduce(1， (a， b) -> a * b);
System.out.println(product2);   //  120

// 3.计算最大值和最小值
Stream<Integer> stream = Stream.of(11， 23， 13， 34， 27);
Optional<Integer> max = stream.reduce((a， b) -> a > b ? a : b);
System.out.println(max.get());  //  34
Stream<Integer> stream1 = Stream.of(11， 23， 13， 34， 27);
Optional<Integer> min = stream1.reduce((a， b) -> a > b ? b : a);
System.out.println(min.get());  //  11
// 上面的代码一样也可以使用Integer自带的max方法和min方法，和上面求和的代码类似
```

    提示stream has already been operated upon or closed

    Stream<String> stream = Stream.of("hello"，"java"，"world");
stream.forEach(System.out::println);  //  hello java world
System.out.println(stream.count());   //    报错:不能重复操作Stream流或流已关闭

  • Stream流方法返回的是新的流

  • Stream流不调用终结方法，中间的操作不会执行

  • 在Stream流中无法直接修改集合、数组中的数据。

Stream还有一些其他操作方法，例如mapToInt，mapToDouble，mapToLong可以将普通的流转换为特定类型的流，从而使得该流可以使用一些特有的计算方法。

IntStream intStream = stream1.mapToInt(a -> a);
//	方法2	使用IntStream直接创建
IntStream intStream = IntStream.of(1， 2， 3);

# Stream流的收集

把Stream流操作后的结果数据转回到集合或者数组中。

  • Stream流：方便操作集合/数组的手段。

  • 集合/数组：才是开发中的目的。

  1. 将Stream流转换为List集合

    Stream<Integer> stream1 = Stream.of(1， 2， 3， 4， 5);
List<Integer> list = stream1.collect(Collectors.toList());
System.out.println(list);   //  [1， 2， 3， 4， 5]
