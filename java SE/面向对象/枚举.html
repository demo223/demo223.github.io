# 什么是枚举?

Java 枚举是一个特殊的类，一般表示一组常量，比如一年的 4 个季节，一个年的 12 个月份，一个星期的 7 天，方向有东南西北等。

例如定义一个方向的枚举类，有上下左右

enum Direction {
	LEFT, RIGHT, TOP, BOTTOM
}

  public static final direction LEFT;
  public static final direction RIGHT;
  public static final direction TOP;
  public static final direction BOTTOM;
  public static direction[] values();
  public static direction valueOf(java.lang.String);
  static {};
}

# 枚举类的使用

每个枚举都是通过 Class 在内部实现的，且所有的枚举值都是 public static final 的。

enum Direction {
	LEFT, RIGHT, TOP, BOTTOM
}

public class Test {
	public static void main(String[] args) {
		Direction d1 = Direction.LEFT;
		System.out.println(d1); //  LEFT
		Color c1 = Color.RED;
		System.out.println(c1); //  RED
	}

	enum Color {
		RED, BLUE, GREEN
	}
}

# 枚举类的常用用法

操作 方法
以数组形式返回枚举类型的所有成员(返回结果为枚举类型对象) values()
将普通字符串转换为枚举实例 valueOf(枚举成员字符串形式)
比较两个枚举成员在定义时的顺序(返回结果表示相差的索引值) 枚举成员.compareTo(枚举成员)
获取枚举成员的索引位置 枚举成员.ordinal()
enum Color {
	RED, BLUE, GREEN
}

System.out.println("以数组形式返回枚举类型的所有成员");
Color[] colors = Color.values();
System.out.println(Arrays.toString(colors));    //  [RED, BLUE, GREEN]

System.out.println("将普通字符串转换为枚举实例");
String str = "RED";
Color color = Color.valueOf(str);
System.out.println(color);  //  RED

System.out.println("比较两个枚举成员在定义时的顺序");
Color color1 = Color.BLUE;
Color color2 = Color.GREEN;
int res = color1.compareTo(color2);
System.out.println(res);    //  -1

System.out.println("获取枚举成员的索引位置");
Color c = Color.BLUE;
int index = c.ordinal();
System.out.println(index);  //  1

# 枚举类型在switch case中的使用

public class Test1 {
	public static void main(String[] args) {
		// 枚举类型在switch case中的使用
		getColor(Color.BLUE);   //  输出蓝色
	}

	public static void getColor(Color c) {
		switch (c) {
			case RED:
				System.out.println("输出红色");
				break;
			case GREEN:
				System.out.println("输出绿色");
				break;
			case BLUE:
				System.out.println("输出蓝色");
				break;
			default:
				System.out.println("输出颜色错误");
		}
	}

	enum Color {
		RED, BLUE, GREEN
	}
}

# 枚举类特点

枚举跟普通类一样可以用自己的变量、方法和构造函数，构造函数只能使用 private 访问修饰符，所以外部无法调用。

枚举既可以包含具体方法，也可以包含抽象方法。 如果枚举类具有抽象方法，则枚举类的每个实例都必须实现它。

  • 枚举类都是继承了枚举类型：java.lang.Enum
  • 枚举都是最终类，不可以被继承。
  • 构造器都是私有的，枚举对外不能创建对象。
  • 枚举类的第一行默认都是罗列枚举对象的名称的。
  • 枚举类相当于是多例模式。
