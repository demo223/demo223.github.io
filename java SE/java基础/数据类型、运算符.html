# 数据类型、运算符

# 数据类型分类

java的数据类型分为基本数据类型和引用数据类型.

基本数据类型分为4大类8大种.

数据类型 取值范围 内存占用
byte -128~127 1
short -32768~32767 2
int -2147483648~2147483647 (10位数) 4
long -9223372036854775808 ~ 9223372036854775807 (19位数) 8
float 1.401298 e^-45 到 3.402823 e^+38 4
double 4.9000000 e^-324 到 1.797693 e^+308 8
char 0-65535 2
boolean true,false 1
  1. byte short int（默认） long 整型
  2. float double（默认） 浮点型
  3. char 字符型
  4. boolean 布尔型

引用数据类型：String，StringBuffer，ArrayList，HashSet，HashMap等。

# 数据类型转换

自动类型转换：类型范围小的变量，可以直接赋值给类型范围大的变量。

byte 、short、char ===> int ===> long ===> float ===> double

注意事项：

  1. 表达式的最终结果类型由表达式中的最高类型决定。
  2. 在表达式中，byte、short、char 是直接转换成int类型参与运算的。

# 逻辑运算符

逻辑与&，逻辑或| ，逻辑非!，逻辑异或^。

逻辑运算符 介绍 说明
& 逻辑与 必须都是true，结果才是true; 只要有一个是false，结果一定是false。
| 逻辑或 只要有一个为true、结果就是true .
! 逻辑非 真变假，假变真
^ 逻辑异或 如果两个条件都是false或者都是true则结果是false。两个条件不同结果是true。

# 短路逻辑运算符

短路逻辑运算符 介绍 说明
&& 短路与 判断结果与&一样。左边为 false，右边则不执行。
|| 短路或 判断结果与|一样。左边为true，右边则不执行。

注意: &&的优先级高于||

# 三元运算符

条件表达式?值1:值2

首先计算关系表达式的值，如果值为true，返回值1，如果为false，返回值2。

# 位运算符

符号 描述 运算规则
& 与 两个位都为1时，结果才为1
| 或 两个位都为0时，结果才为0
^ 异或 两个位相同为0，相异为1
~ 取反 0变1，1变0
<< 左移 各二进位全部左移若干位，高位丢弃，低位补0
>> 右移 各二进位全部右移若干位，对无符号数，高位补0，
有符号数，各编译器处理方法不一样，有的补符号位（算术右移），有的补0（逻辑右移）
int a = 3;  //  二进制011
int b = 4;  //  二进制100
// 与运算符    |   两个运算对象对应位都为1时为1
System.out.println(a & b);  //  0
// 或运算符    |   两个运算对象对应位为0时为0
System.out.println(a | b);  //  7   二进制111
// 异或运算符    ^   两个运算对象对应位相同为0,不同为1
System.out.println(a ^ b);  //  7   二进制111
// 取反运算符    ~   一元运算符   0变1,1变0
System.out.println(~a);     //  -4
System.out.println(~b);     //  -5
// 左移运算符    <<  (a^n)  = a*=2^n
System.out.println(b << 2); //  16  4*2^2 = 16
// 右移运算符    >>  (a^n)  = a/=2^n
System.out.println(b >> 2); //  1   4/2^2 = 1
