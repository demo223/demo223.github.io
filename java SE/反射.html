# 反射

  • 反射是指对于任何一个Class类，在运行的时候都可以直接得到这个类全部成分。
  • 在运行时，可以直接得到这个类的构造器对象：Constructor
  • 在运行时，可以直接得到这个类的成员变量对象：Field
  • 在运行时，可以直接得到这个类的成员方法对象：Method
  • 这种运行时动态获取类信息以及动态调用类中成分的能力称为Java语言的反射机制。

# 获取字节码

反射的第一步都是先得到编译后的Class类对象，然后就可以得到Class的全部成分

# Class.forName("包名.类名")

通过Class的静态类forName()方法获取一个类的字节码

Class<?> clazz1 = Class.forName("com.zq.myStudy.反射.User");
System.out.println(clazz1);	//	class com.zq.myStudy.反射.User

获取非public类型的的Constructor、Field、Method对象

反射的API方法命名，有下面一些规律，自己体会一下。

  • 后缀带s的，表示获取多个返回一个数组对象；后缀不带s表示获取其中指定一个
  • 方法名带有Declared的，表示可以获取私有修饰的成员
  • 在使用非public类型的Constructor、Field、Method对象的方法之前，需要使用(Constructor、Field、Method)对象.setAccessible(true);将私有权限取消，再进行调用或赋值

例如上面的完整代码，有如下测试类User

		System.out.println("私有方法");

采用反射的方式，创建Student对象，并给name和age赋值，然后获取name和age的值

public class Test {
    public static void main(String[] args) throws Exception {
        Class<Student> clazz = Student.class;
        // 获取一个无参构造器
        Constructor<Student> constructor = clazz.getConstructor();
        // 根据无参构造器创建一个对象
        Student student = constructor.newInstance();
        // 获取对象的公有属性name
        Field name = clazz.getField("name");
        // 获取对象的私有属性age
        Field age = clazz.getDeclaredField("age");
        // 设置name对象的属性值
        name.set(student,"张三");
        // 设置age对象的属性值
        // 由于age是私有属性,必须先开放权限,才能进行设置
        age.setAccessible(true);
        age.set(student,23);

        // 获取对象student的name属性值
        String value1 = (String) name.get(student);
        // 获取对象student的age属性值
        Integer value2 = (Integer) age.get(student);
        System.out.println("public属性name的值:"+value1);   //  public属性name的值:张三
        System.out.println("private属性age的值:"+value2);   //  private属性age的值:23
    }
}

# 破坏泛型的约束

应用举例：绕过编译阶段为集合添加数据

例如有如下一个ArrayList<Integer>的集合

public class Test {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        list.add(12);
        list.add("hello");  //  编译报错
        list.add(23);
    }
}

泛型只是在编译阶段可以约束集合只能操作某种数据类型，在编译成Class文件进入运行阶段的时候，其真实类型都是ArrayList了(上面例子中)，泛型相当于被擦除了。

反射是作用在运行时的技术，此时集合的泛型将不能产生约束了，此时是可以为集合存入其他任意类型的元素 的。

public class Test8 {
    public static void main(String[] args) throws Exception {
        ArrayList<Integer> list1 = new ArrayList<>();
        ArrayList<String> list2 = new ArrayList<>();

        System.out.println(list1.getClass());   //  class java.util.ArrayList
        System.out.println(list2.getClass());   //  class java.util.ArrayList
        System.out.println(list1.getClass() == list2.getClass());   //  true
        // 可以发现泛型被擦除了都是ArrayList类型的了

        // 获取ArrayList集合的字节码
        Class<? extends ArrayList> clazz = list1.getClass();
        ArrayList arrayList = clazz.getConstructor().newInstance();
        arrayList.add(1);
        arrayList.add("hello");
        System.out.println(arrayList);  //  [1, hello]

        // 绕过类型判断另一种做法(默认泛型是Object类型的)
        ArrayList list3 = new ArrayList<>();
        list3.add(1);
        list3.add("hello");
        System.out.println(list3);  //  [1, hello]
    }
}

# 反射做通用框架

需求：给你任意一个对象，在不清楚对象字段的情况下，可以把对象的字段名称和对应值存储到文件中去。

分析

  1. 定义一个方法，可以接收任意类的对象。

  2. 每次收到一个对象后，需要解析这个对象的全部成员变量名称。

  3. 这个对象可能是任意的，那么怎么样才可以知道这个对象的全部成员变量名称呢？

  4. 使用反射获取对象的Class类对象，然后获取全部成员变量信息。

  5. 遍历成员变量信息，然后提取本成员变量在对象中的具体值

  6. 存入成员变量名称和值到文件中去即可。

创建一个Framework类，定义一个save方法接收任意类型的对象

public class Framework {
	//	保存任意对象
	public static void save(Object obj) {
		try {
			// 创建打印流对象
			PrintStream ps = new PrintStream(new FileOutputStream("src/反射文件.txt"));
			// 打印obj对象的简单名称
			ps.println("-----------" + obj.getClass().getSimpleName() + "-----------");
			// 获取字节码
			Class<?> clazz = obj.getClass();
			// 获取对象所有的成员变量(包括私有的)
			Field[] fields = clazz.getDeclaredFields();
			// 获取成员变量的信息
			for (Field field : fields) {
				// field.getName()返回Field对象表示的字段的名称
				String name = field.getName();
				field.setAccessible(true);  //  取消私有访问权限
				String value = (String) field.get(obj);
				ps.println(name + "=" + value);
			}
		} catch (FileNotFoundException e) {
			throw new RuntimeException(e);
		} catch (IllegalAccessException e) {
			throw new RuntimeException(e);
		}
	}
}

读取配置文件，再通过反射来执行配置文件中className对应的类中的methodName方法

public class Test1 {
	public static void main(String[] args) throws Exception {
		Properties props = new Properties();
		props.load(new FileInputStream("src/com/zq/myStudy/反射/demo3/config.properties"));
		Class<?> clazz = Class.forName(props.getProperty("className"));
		String methodName = props.getProperty("methodName");
		Method method = clazz.getDeclaredMethod(methodName);
		method.setAccessible(true);
		method.invoke(clazz.getConstructor().newInstance());    //  学生在学习
	}
}
