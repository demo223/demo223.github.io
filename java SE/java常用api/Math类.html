# Math类

# math类的常用方法

Java 的 Math 包含了用于执行基本数学运算的属性和方法，如初等指数、对数、平方根和三角函数。

Math 的方法都被定义为static形式，通过 Math 类可以在主函数中直接调用。

参数一般为(byte,short,int,long, float, double)这6种类型.

  1. 返回给定数据的绝对值 Math.abs()

    System.out.println(Math.abs(-1));   //  1
System.out.println(Math.abs(-123L));    //  123
System.out.println(Math.abs(-1.1)); //  1.1
System.out.println(Math.abs(-1.123));   //  1.123

    2

    4

  2. 向上取整 Math.ceil()

    System.out.println(Math.ceil(1.2f));    //  2.0
System.out.println(Math.ceil(1.2));     //  2.0
System.out.println(Math.ceil(-1.2));    // -1.0
    1
    2
    3

  3. 向下取整 Math.floor()

    System.out.println(Math.floor(1.2f));   //  1.0
System.out.println(Math.floor(1.2));    //  1.0
System.out.println(Math.floor(-1.2));   // -2.0
    1
    2
    3

  4. 四舍五入 Math.round()

    System.out.println(Math.round(1.2f));   //  1
System.out.println(Math.round(-1.2));   //  -1
    1
    2

    注意: 返回值是long类型

  5. 返回2个数中最大的值 Math.max(a,b)

    System.out.println(Math.max(1, 2));  //  2
System.out.println(Math.max(1, 2.2));  //  2.2
System.out.println(Math.max(-1.1, 2.2));  //  2.2
    1
    2
    3

  6. 返回2个数中最小的值 Math.min(a,b)

    System.out.println(Math.min(1, 2));  //  1
System.out.println(Math.min(1, 2.2));  //  1.0
System.out.println(Math.min(-1.1, 2.2));  //  -1.1
    1
    2
    3

  7. 返回第一个参数的第二个参数次方 Math.pow(a,b)

    System.out.println(Math.pow(2, 3));   //  8.0
System.out.println(Math.pow(2, -1));  //  0.5
System.out.println(Math.pow(2, 3.0));  //  8.0
    1
    2
    3

    注意: 返回值是double类型的

  8. 返回参数的算术平方根 Math.sqrt()

    System.out.println(Math.sqrt(4));   //  2.0
    1

    注意: 返回值是double类型

  9. 返回一个随机数 Math.random() [0,1.0)之间的一个随机double数据

    System.out.println(Math.random());  //  0.9148539399370038
System.out.println(Math.random());  //  0.6518956296792914
    1
    2

  10. 返回与参数最接近的整数。返回类型为double。 Math.rint()

    System.out.println(Math.rint(1.1));   //  1.0
System.out.println(Math.rint(-1.1));  //  -1.0
System.out.println(Math.rint(1.5));   //  2.0
System.out.println(Math.rint(-1.5));  //  -2.0
    1
    2
    3
    4

    关于Math.round与Math.rint区别

    1. Math.round的返回值为long类型,Math.rint返回值类型为double类型.

    2. 取整规则不同

      • Math.round是我们所知道的四舍五入

      • Math.rint一般情况和Math.round的返回值没有什么区别,唯一的区别是在X.5的边界上根据X的奇偶性来决定返回值.当X为奇数的时候返回值,X为偶数的时候向下取整.

        System.out.println(Math.rint(1.5)); //  2.0
System.out.println(Math.rint(2.5)); //  2.0
System.out.println(Math.rint(-1.5));    //  -2.0
System.out.println(Math.rint(-2.5));    //  -2.0
        1
        2
        3
        4

        可以看出Math.rint的返回值都是偶数,所以可以简单的记为当Math.rint的参数为X.5这样处于2个整数中间样式时,Math.rint返回与它最接近的偶数.

  11. 剩下的不常用的Math方法有:

    1. Math.sin()返回指定参数的正弦值
    2. Math.cos()返回指定参数的余弦值
    3. Math.tan()返回指定参数的正切值
    4. Math.asin()返回指定参数的反正弦值
    5. Math.acos()返回指定参数的反余弦值
    6. Math.atan()返回指定参数的反正切值
    7. Math.exp()返回自然底数e的参数次方
    8. Math.log()返回参数的自然数底数的对数值
