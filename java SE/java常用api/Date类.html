# SimpleDateFormat类

对于输出格式，日期和时间模式(注意大小写，代表的含义是不同的)

yyyy：年 MM：月 dd：日 hh：1~12小时制(1-12) HH：24小时制(0-23) mm：分 ss：秒 S：毫秒 E：星期几 D：一年中的第几天 F：一月中的第几个星期(会把这个月总共过的天数除以7) w：一年中的第几个星期 W：一月中的第几星期(会根据实际情况来算) a：上下午标识 k：和HH差不多，表示一天24小时制(1-24)。 K：和hh差不多，表示一天12小时制(0-11)。 z：表示时区

创建SimpleDateFormat日期格式化器(SimpleDateFormat对象)

  1. 日期格式化：将一个日期对象Date转换为字符串String

    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH-mm-ss");
System.out.println(sdf.format(new Date())); //  2022-10-15 09-50-20

SimpleDateFormat sdf2 = new SimpleDateFormat("yyyy年MM月dd日 HH时mm分ss秒");
System.out.println(sdf2.format(new Date()));   //  2022年10月15日 10时00分50秒

  2. 日期解析：将字符串String转换为日期对象Date

    String str = "2022-10-15 09-54-20";
Date date2 = sdf.parse(str);
System.out.println(date2);  //  Sat Oct 15 09:54:20 CST 2022

# Calendar类

Calendar类是一个抽象类，不能直接创建对象，需要通过getInstance获取一个Calendar类的实例对象

Calendar cal = Calendar.getInstance();
1

通过getTime()来获取日历类的时间对象

cal.getTime()	//	返回一个Date对象
1

  • 使用set()方法来进行设置指定时间

    cal.setTime(new Date());
cal.set(2022,10,15);
cal.set(2022,10,15,10,55);
cal.set(2022,10,15,10,55,58);
cal.set(Calendar.YEAR, 2022);//年    
cal.set(Calendar.MONTH, 9);//月（月份0代表1月）    
cal.set(Calendar.DATE, 15);//日    
cal.set(Calendar.HOUR_OF_DAY, 10);//时
