# Number类

一般地，当需要使用数字的时候，我们通常使用内置数据类型，如：byte、int、long、double 等。

int a = 132;
double b = 12.35;
1
2

然而，在实际开发过程中，我们经常会遇到需要使用对象，而不是内置数据类型的情形。为了解决这个问题，Java 语言为每一个内置数据类型提供了对应的包装类。Number类属于 java.lang 包。

所有的包装类**（Integer、Long、Byte、Double、Float、Short）**都是抽象类 Number的子类。

包装类 基本数据类型
Byte byte
Short short
Integer int
Long long
Float float
Double double
Boolean boolean
Character char

System.out.println(a.floatValue());  // 113.0
System.out.println(a.doubleValue());  // 113.0

  3. 将Number对象和参数进行比较 Number对象.compareTo(参数)

    该方法用于两个相同数据类型的比较，两个不同类型的数据不能用此方法来比较。

System.out.println(a.compareTo(100));   //  1
