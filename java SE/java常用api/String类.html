# String类

在 Java 中字符串属于对象，Java 提供了 String 类来创建和操作字符串。

# 创建字符串

创建字符串有2种方式

  1.  String s1 = "hello";
    1
  2.  String s2 = new String("hello");
    1

注意: s1和s2是不同的对象

System.out.println(s1 == s2);   //  false
1

# 字符串长度

String 类的一个访问器方法是 length() 方法，它返回字符串对象包含的字符数。注意要加().

数组获取数组的长度使用的是数组的length属性,是不加()的.

System.out.println("hello".length());   //  5
1

    使用字符串字面量和new关键字之间的区别

    在Java中，JVM维护一个字符串池来将其所有字符串存储在内存中。字符串池有助于重用字符串。

    使用字符串文字创建字符串时，直接提供字符串的值。因此，编译器首先检查字符串池，看看字符串是否已经存在。

    • 如果字符串已经存在， 不会创建新字符串。相反，新引用指向现有字符串。
    • 如果字符串不存在，则创建新字符串。

    但是，在使用new关键字创建字符串时，不会直接提供字符串的值。因此，新字符串总是被创建。

    引用举例：

    String s1 = new String("hello");
String s2 = "hello";
String s3 = "he" + "llo";
String s4 = "he";
String s5 = "llo";
System.out.println(s1 == s2);   //  false
System.out.println(s3 == s2);   //  true
System.out.println(s4 + s5 == s2);  //  false
    1
    2
    3
    4
    5
    6
    7
    8

    解释：

    • 两个或者两个以上的字符串常量相加，在预编译的时候+会被优化，相当于把两个或者两个以上字符串常量自动合成一个字符串常量

    • 字符串的+操作其本质是new了StringBuilder对象进行append操作，拼接后调用toString()返回String对象
